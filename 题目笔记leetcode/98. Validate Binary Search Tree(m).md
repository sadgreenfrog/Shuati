## 
98. Validate Binary Search Tree

https://leetcode.com/problems/validate-binary-search-tree/

My Solution

```python
# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
#----------------迭代------------------------#
class Solution:
    def isValidBST(self, root: TreeNode) -> bool:
        if not root: return False
        st,res = [],[]
        cur = root
        while cur or st:
            if cur:
                st.append(cur)
                cur = cur.left
            else:
                cur = st.pop()
                res.append(cur.val)
                cur = cur.right         
        l = len(res)
        for i in range(1, l):
            if res[i] <= res[i-1]:
                return False
        return True
        
```
## 解题思路
迭代法:中序遍历 + 判断是否是递增数组

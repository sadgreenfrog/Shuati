## Find First and Last Position Of Element in Sorted Array

https://leetcode.com/problems/find-first-and-last-position-of-element-in-sorted-array/

My Solution

```python
class Solution(object):
    def searchRange(self, nums, target):
        a = len(nums)
        l, r = 0, a-1
        pos1, pos2 = -1, -1
        while(l<=r):            
            mid = l + (r-l)//2
            if target < nums[mid]:
                r = mid - 1;
            elif target > nums[mid]:
                l = mid + 1;
            else:
                pos1 = mid
                r = mid - 1
                
        l, r = 0, a-1 
        while(l<=r):            
            mid = l + (r-l)//2
            if target < nums[mid]:
                r = mid - 1;
            elif target > nums[mid]:
                l = mid + 1;
            else:
                pos2 = mid
                l = mid + 1      
        return [pos1, pos2]
```

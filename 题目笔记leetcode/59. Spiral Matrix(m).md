## 59. Spiral Matrix II

https://leetcode.com/problems/minimum-size-subarray-sum/

My Solution

```python
class Solution(object):
    def minSubArrayLen(self, target, nums):
        """
        :type target: int
        :type nums: List[int]
        :rtype: int
        """
        if not nums:
            return 0
        l,r = 0, 0
        a = len(nums)
        sum = 0
        ans = a + 1
        while r < a:
            sum = sum + nums[r]
            while(sum >= target):
                ans = min(ans, r-l+1)
                sum = sum - nums[l]
                l = l + 1
            r = r + 1
        return 0 if ans == a+1 else ans
```
